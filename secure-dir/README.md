# Các tính năng cho USER
- Xem và sửa file trong open-dir
- Hết rồi bạn còn muốn gì nữa
# Các tính năng cho ADMIN
## AddUser
- Thêm tài khoản
```
AddUser <username> <password> <role>
```
## DelUser
- Xóa tài khoản
```
DelUser. <username>
```
## ask_AI
- Gọi API của gemini để kiểm tra vấn đề bảo mật qua log
```
ask_AI
```
